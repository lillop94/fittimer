Title: Settings
Description: Testing the settings, music and sounds
Preconditions: Open app https://lillop94.github.io/fittimer/

 1. Click button "Налаштування"
 2. Select toggle on "Звукові сигнали", "Сигнал на середині"
 3. Select "Гучність сигналів" to 90%, "Гучність музики" to 50%
 3. Back to main menu and click button "Тренування"
 4. Select and start any workout
 5. Done or skip trainig
 6. Back to "Налаштування"
 7. Select toggle off "Звукові сигнали", "Сигнал на середині"
 8. Select "Гучність сигналів" to 20%, "Гучність музики" to 100%
 9. Open menu "Тренування"and start any workout
 10. Done trainig

 Expected result: Sounds and music are change volume. When toggles on, in workouts has additional sounds.
